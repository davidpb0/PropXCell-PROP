Atajos
A continuación encontrará el listado de atajos de teclado disponibles en nuestro sistema.

*********
Documento
*********

- Ctrl + N : Nuevo documento
- Ctrl + O : Abrir documento
- Ctrl + S : Guardar documento
- Alt + N : Nueva hoja

*********
Edición
*********

- Ctrl + C : Copiar
- Ctrl + V : Pegar
- Ctrl + X : Cortar

*********
Formato
*********

- Ctrl + B : Negrita
- Ctrl + I : Cursiva
