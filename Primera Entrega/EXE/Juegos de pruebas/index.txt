## Elementos del directorio:

### DescripcionJuegosDePruebas.txt
Este archivo contiene la descripcion para el uso y entendimiento de los juegos de pruebas empleados
para testear las funcionalidades del programa.

### Importame.csv
Archivo de prueba en formato csv para que se importe en una hoja con finalidad de testeo.

### Juegos de pruebas.prop
Este archivo se trata de un Documento guardado, el cual podemos cargar desde la aplicacion para
observar los juegos de pruebas.
