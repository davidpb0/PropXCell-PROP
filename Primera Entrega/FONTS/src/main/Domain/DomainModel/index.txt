## Elementos del directorio

### AproximarValor:
Esta clase es utilizada para la funcion "Aproximar Valor" de la hoja de calculo.

### BloqueSeleccionado:
Esta clase contiene una celda inicial y una celda final, que son las celdas que definen el bloque con forma de
rectangulo. Tambien contiene su hoja para saber donde se ubican las celdas.

### BloqueTemporalCopiado:
Esta clase se guarda una matriz de celdas para usarse como portapapeles y poder pegarse donde se quiera(dentro del
software).

### Celda
Esta clase es la unidad atomica de informacion y es capaz de guardar valores, resultados de funciones y referencias
a otras celdas.
