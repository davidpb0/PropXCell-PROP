## Elementos del directorio

### DOCS:

Contiene la documentacion del proyecto.

### EXE:

Ejecutables de cada clase, driver y test unitario, cada uno en un subdirectorio.

### FONTS:

Clases implementadas por el equipo, tanto clases de dominio como controladores, drivers y tests unitarios. Todas
las clases son compilables con un Makefile.

## Ejecutar las clases

Para ejecutar las diferentes clases implementadas hay que usar el Makefile que se encuentra en /FONTS/src/, primero
utilizando la opcion "make build", y posteriormente "make help" para consultar las opciones disponibles.
