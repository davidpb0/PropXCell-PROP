## Elementos del directorio:

### imagenes:
Directorio que contiene las imagenes utilizadas para iconos, etc.

### vistas:
Package que contiene las vistas del programa.

### ControladorPresentacion:
Este controlador, singleton, se guarda una instancia de la PantallaInicial para iniciarla cuando toque.

### MainProgram:
Clase que contiene el metodo Main del programa.
