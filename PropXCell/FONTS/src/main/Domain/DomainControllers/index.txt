## Elementos del directorio

### ControladorBloque
Este controlador engloba las funcionalidades que puede ejecutar un bloque seleccionado.

### ControladorDocumento
Este controlador engloba las funcionalidades que puede ejecutar el documento.

### ControladorDominio
Este controlador es singleton y contiene una instancia de cada controlador de la capa de dominio.

### ControladorHoja
Este controlador engloba las funcionalidades que puede ejecutar una hoja.
